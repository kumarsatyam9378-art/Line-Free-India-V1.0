✅ Task 10.1 Implementation Details

1. Page Entrance Animation

Main container fades in and slides up from 20px below

initial: { opacity: 0, y: 20 }
animate: { opacity: 1, y: 0 }
duration: 0.4s, easing: cubic-bezier(0.4, 0, 0.2, 1)

2. Fade-in Animation

Header section fades in smoothly

initial: { opacity: 0 }
animate: { opacity: 1 }
duration: 0.3s, delay: 0.1s

3. Slide-up Animation

Content sections slide up from 20px below

initial: { opacity: 0, y: 20 }
animate: { opacity: 1, y: 0 }
duration: 0.3s, easing: cubic-bezier(0, 0, 0.2, 1)

4. Stagger Animation for Business Cards

Cards appear sequentially with 60ms delay between each

initial: { opacity: 0, y: 30, scale: 0.95 }
animate: { opacity: 1, y: 0, scale: 1 }
staggerChildren: 0.06s (60ms)
delayChildren: 0.1s
duration: 0.4s, easing: cubic-bezier(0, 0, 0.2, 1)

5. Hover Animations

Business cards scale up and lift on hover

hover: { scale: 1.02, y: -4px }
duration: 0.2s
